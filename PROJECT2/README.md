# Student Performance Prediction System

## Project Description
The Student Performance Prediction System is a Machine Learning project that predicts a student's final performance based on Attendance, Study Hours, and Internal Marks. The project uses a Linear Regression model to estimate the expected final marks and helps analyze student performance.

## Features
- Student performance prediction
- CSV file handling
- Data preprocessing
- Machine Learning using Linear Regression
- Data visualization with charts
- Model accuracy evaluation
- User input for prediction

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Google Colab

## Dataset
The dataset contains the following attributes:
- Attendance
- Study Hours
- Internal Marks
- Final Result

## Machine Learning Model
- Linear Regression

## How to Run
1. Open the notebook in Google Colab.
2. Run all the code cells.
3. Train the machine learning model.
4. Enter Attendance, Study Hours, and Internal Marks.
5. View the predicted final marks.
6. Generate the visualization chart.
7. Check the model accuracy.

## Output
The system predicts the student's final marks based on the input values and displays the model accuracy and visualization graph.
