### PROJECT 5

# Smart Recommendation System

## Project Description
The Smart Recommendation System is a Machine Learning project that recommends similar movies based on the user's preference. It uses a content-based filtering approach with Cosine Similarity to analyze movie genres and generate personalized recommendations.

## Features
- User preference input
- Personalized movie recommendations
- Content-based filtering
- Cosine Similarity algorithm
- CSV file handling
- Fast and accurate recommendation system

## Technologies Used
- Python
- Pandas
- Scikit-learn
- CountVectorizer
- Cosine Similarity
- Google Colab

## Dataset
The dataset contains:
- Movie Name
- Movie Genre

## Machine Learning Algorithm
- Content-Based Filtering
- CountVectorizer
- Cosine Similarity

## How to Run
1. Open the notebook in Google Colab.
2. Run all the code cells.
3. Enter the name of a movie.
4. The system will recommend similar movies based on genre similarity.

## Sample Input
Movie Name:
Avengers

## Sample Output
Recommended Movies:
- Iron Man
- Doctor Strange
- Batman
