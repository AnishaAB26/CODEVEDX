# Utility Usage Prediction Tool

## Project Description
This is a machine learning-based utility usage prediction tool developed using Python. It predicts future utility usage based on historical monthly data.

## Features
- View utility data
- Add new records
- Update existing records
- Predict utility usage
- CSV file handling
- Exception handling
- Linear Regression Machine Learning model

## Technologies Used
- Python
- Pandas
- Scikit-learn
- CSV

## How to Run
1. Open the project in Google Colab or Python.
2. Run all the cells.
3. Select an option from the menu.
4. Enter the required data.
